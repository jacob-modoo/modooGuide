

# 메세지단 개발

`목적`

#### 사용자간 원할한 인스턴트 메세지 남기기. 

> 예외 사항이 있는 사용자 없이 모든 사용자의    
> 커뮤니 케이션이 가능 하도록 만든다. 

## 기능

>  1. 메세지
>  2. 이모티콘
>  3. 다른 채팅방으로의 이동
>  4. 채팅방 푸쉬 알림 안받기. 
>  5. 참여중인 사용자 보기.
>6. 공지 사항 띠우기. 기능
>  7. 이미지 보내기 기능. 
>
