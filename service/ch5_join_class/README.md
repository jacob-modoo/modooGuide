## 모두의 클래스 서비스 개요 

>  모두의 클래스 서비스는 누구나 코치가 되어 온라인 클래스를 열어 수익을 가져 갈 수 있는 플랫폼형 웹 / 앱 서비스로 크게 아래 7개 장으로 구성되어 있다.
>
> 각 장은 화면단 `구성|기능|디자인|작업자 `  로  되어 있으며, 각 장은 추가 개선사항과  Configuration 문서로 연결 되어 관리한다.
