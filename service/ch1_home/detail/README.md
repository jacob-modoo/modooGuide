	- 본 클래스는 어떻게 진행되나요?

	- 클래스 추천대상
		> 클래스 개설시 입력한 내용
	- 코치 소개
		> 제목 | 코치사진 | 코치소개글 (sns url 포함) | 더 보기/접기
	- 인터뷰
